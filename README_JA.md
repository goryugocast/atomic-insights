# Adamic Adar Graph

思考を育てる人のための、知的バックアップ・ツール。

Obsidianでアトミックなノート（一つの概念に絞った小さなメモ）を作り、それらをリンクで繋いでいく「Andy Matuschak氏のエバーグリーン・ノート」の実践者にとって、最適な関連ノート表示プラグインです。

## コンセプト
単なるキーワードの一致や「同じタグ」といった表面的な繋がりではなく、**Adamic Adar（アダミック・エイダ）**というアルゴリズムを用いて、グラフ構造の中から「深い思考の繋がり」を掘り起こします。

### なぜ「Adamic Adar」なのか？
もし2つのノートが、「誰もがリンクしている有名なノート」を共有していても、その関連性は偶然かもしれません。
しかし、**「ごく一部のノートしかリンクしていない、マニアックなノート」を共有しているとしたら？** そこには強い思考の繋がりがあるはずです。

Adamic Adarは、こうした「珍しい共通の繋がり」を重く評価します。あなたが精緻なナレッジベースを構築すればするほど、AIには真似できない「あなた自身の思考のクセ」に基づいた、意外なノート同士の再会を提供してくれます。

## 主な機能
- **Adamic Adar アルゴリズム特化**: 思考の深まりに最適な計算手法を採用。
- **フォルダ除外設定**: テンプレートやアーカイブ、デイリーノートなど、計算に含めたくないフォルダを指定可能。
- **高速・軽量・堅牢**: 以前のプラグインにあった重厚な依存関係を排除し、Obsidian標準の挙動に準拠。
- **ネイティブな操作感**: サイドバーでの常駐、クリックによるノート移動、ドラッグ＆ドロップによるリンク作成に対応。

## 使い方
1. コマンドパレットから `Open Graph Analysis View` を実行。
2. 右サイドバーに現れるビューが、現在開いているノートに合わせて自動更新されます。
3. リスト内の項目をクリックして移動、またはエディタにドラッグしてリンクを作成できます。
4. プログラムの設定（Settings）から、計算から除外したいフォルダを1行ずつ指定できます。

## インストール
(Obsidian Community Plugins への登録準備中)

### 手動インストール
1. `main.js`, `manifest.json`, `styles.css` をダウンロードします。
2. あなたの保管庫（Vault）のプラグインフォルダ（例: `.obsidian/plugins/adamic-adar-graph/`）にコピーします。
3. 設定からプラグインを有効化してください。
